# Data Analyzer and Transformer Program

## Project Description
The Data Analyzer and Transformer Program is a menu-driven Python application that allows users to perform various operations on a dataset. The program demonstrates the use of Python built-in functions, recursion, lambda functions, sorting techniques, and functions that return multiple values.

---

## Features

### 1. Input Data
- Accepts a 1D array of integers from the user.
- Stores the data for further processing.

### 2. Display Data Summary (Built-in Functions)
Displays:
- Total number of elements
- Minimum value
- Maximum value
- Sum of all values

### 3. Calculate Factorial (Recursion)
- Calculates the factorial of a given number using a recursive function.

### 4. Filter Data by Threshold (Lambda Function)
- Filters and displays values greater than or equal to a user-defined threshold using a lambda function.

### 5. Sort Data
Provides two sorting options:
- Ascending Order
- Descending Order

### 6. Display Dataset Statistics (Return Multiple Values)
Displays:
- Minimum value
- Maximum value
- Sum of all values
- Average value

### 7. Exit Program
- Terminates the application safely.

## Program Flow

1. User enters dataset.
2. User selects an operation from the menu.
3. Program processes the selected task.
4. Results are displayed.
5. Menu reappears until Exit is selected.
