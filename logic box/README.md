# 🔷 LogicBox – Pattern Generator & Number Analyzer

A beginner-friendly and interactive Python console project that combines **pattern generation** and **number analysis** using loops, conditions, and menu-driven programming.

This project is designed for students and Python beginners to practice:

- Loops
- Conditional Statements
- Nested Loops
- User Input Handling
- Menu-Driven Programs
- Basic Number Logic

---

# 🚀 Project Overview

**LogicBox** is a simple yet powerful Python program that allows users to:

✅ Generate star (`*`) patterns  
✅ Analyze numbers as Even or Odd  
✅ Calculate the sum of a range of numbers  
✅ Interact using a menu-driven interface  

The project demonstrates core Python fundamentals in a practical and interactive way.

# 🛠️ Technologies Used

- 🐍 Python 3
- While Loops
- For Loops

- Conditional Statements (`if-elif-else`)

# 📌 Features

## ⭐ Pattern Generator

Generate triangle star patterns dynamically based on user input.

### Example Output

```python
*
**
***
****
```

---

## 🔢 Number Analyzer

Analyze numbers in a given range and identify whether each number is:

- Even
- Odd

Also calculates the total sum of all numbers in the range.

### Example Output

```python
1 is Odd
2 is Even
3 is Odd
4 is Even

Sum of numbers = 10
```

---

## 📋 Menu-Driven Program

Interactive menu system for easy navigation.

### Menu

```python
1. Generate a Pattern
2. Analyze Numbers
3. Exit
```

# 💻 Program Workflow

## 1️⃣ Generate a Pattern
- User enters number of rows
- Program prints star pattern using nested loops

## 2️⃣ Analyze Numbers
- User enters starting and ending numbers
- Program checks:
  - Even numbers
  - Odd numbers
- Calculates total sum

## 3️⃣ Exit
- Safely exits the program
